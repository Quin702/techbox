# TechBox

TechBox is a Python application that allows users to manage and edit startup items on Windows to improve boot times. It provides a command-line interface to list, add, and remove startup programs from the registry and the startup folder.

## Features

- **List Startup Items**: Display all startup items currently configured in the registry and startup folder.
- **Add Startup Item**: Add a new program to the startup list by specifying a name and command.
- **Remove Startup Item**: Remove an existing program from the startup list using its name.

## Requirements

- Python 3.x
- Windows Operating System

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/yourusername/techbox.git
   ```

2. Navigate to the project directory:

   ```bash
   cd techbox
   ```

3. Run the application:

   ```bash
   python techbox.py
   ```

## Usage

1. **List Startup Items**: Select option 1 to view all startup items.
2. **Add Startup Item**: Select option 2, provide the name and command to add a new startup item.
3. **Remove Startup Item**: Select option 3, provide the name of the item to remove it from startup.
4. **Exit**: Select option 4 to exit the application.
